# MusicPlayer

Music player submodule for [LyricsX](https://github.com/ddddxxx/LyricsX).

## Supported Player

#### macOS

- [x] Apple Music (iTunes)
- [x] Spotify
- [x] Vox
- [x] Audirvana
- [x] Swinsian

#### iOS

- [x] Music
- [ ] Spotify (blocked by SE-0272)
